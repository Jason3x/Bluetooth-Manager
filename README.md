# Bluetooth-Manager
Bluetooth Manager is a multilanguage Bash script designed for ArkOS-based handheld consoles. 
It provides a simple menu interface to install, configure and manage Bluetooth audio and controllers reliably.
- Enable or disable Bluetooth
- Scan and connect a device
- Display known devices
- Remove a known device
  
# Internet connection 
An internet connection is required for first-time installation

# Languages
- English
- Français
- Español
- Português
- Italiano
- Deutsch
- Polish

# Installation
Place the Bluetooth Manager script in the tools directory of roms/tools or roms2/tools if you have a 2 SD card configuration.
